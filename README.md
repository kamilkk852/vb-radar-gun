# Volleyball Radar Gun project

This project aims to develop a simple mobile application that uses AI technology to estimate the speed of a volleyball. By analyzing a high-speed video captured by a mobile phone camera and knowing the diameter of the volleyball, the app will be able to estimate the speed of the ball.
